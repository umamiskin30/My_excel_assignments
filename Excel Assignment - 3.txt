1. How and when to use the AutoSum command in excel?
- If you want to calculate the cells you can simply go to 'Home' see the option 'Autosum' in the toolbar, click on it and calculate the columns and rows you want.


2. What is the shortcut key to perform AutoSum?
- (Alt + =)


3. How do you get rid of Formula that omits adjacent cells?
- Go to 'File', then go to 'Options' and then select Formulas. Look for Error checking rules and uncheck Formulas which omit cells in a region and click OK.


4. How do you select non-adjacent cells in Excel 2016?
-  Hold Ctrl and select the cells.


5. What happens if you choose a column, hold down the Alt key and press the letters ocw in quick succession?
-  It show the popup of column width where we can customize/change the width of selected column.


6. If you right-click on a row reference number and click on Insert, where will the row be added?
-  Above the selected row.
